# Yilin's Personal Dictionary

This project is a GitHub Pages-ready static site.

## Files

- `index.html`: page structure and content
- `styles.css`: layout and visual design
- `script.js`: navigation highlighting and mobile menu behavior

## Publish with GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the main branch and the `/ (root)` folder.
6. Save, then wait for GitHub Pages to publish the site.

If this repository is named `<your-username>.github.io`, it will publish as your main personal site. Otherwise, it will publish as a project page.
